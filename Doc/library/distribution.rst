***********************************
Software Packaging and Distribution
***********************************

These libraries help you with publishing and installing Herthon software.
While these modules are designed to work in conjunction with the
`Herthon Package Index <https://pypi.org>`__, they can also be used
with a local index server, or without any index server at all.
