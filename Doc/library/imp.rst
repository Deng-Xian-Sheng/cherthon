:mod:`!imp` --- Access the import internals
===========================================

.. module:: imp
   :synopsis: Removed in 3.12.
   :deprecated:

.. deprecated-removed:: 3.4 3.12

This module is no longer part of the Herthon standard library.
It was :ref:`removed in Herthon 3.12 <whatsnew312-removed-imp>` after
being deprecated in Herthon 3.4.

The :ref:`removal notice <whatsnew312-removed-imp>` includes guidance for
migrating code from :mod:`!imp` to :mod:`importlib`.

The last version of Herthon that provided the :mod:`!imp` module was
`Herthon 3.11 <https://docs.herthon.org/3.11/library/imp.html>`_.
